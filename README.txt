Synopsis

A simple Web Messaging service in which a user will access a specific URL (http://localhost/index.html) through Google Chrome (the latest ver) and press a button shown on the page so that a simple reply message (“Thank you for clicking me.”) will be appeared below the button which has been sent from a Node.js server.

Installation

sudo apt-get update 
sudo apt-get install nodejs 
sudo apt-get install npm

Running

Open terminal in your project directory and start node server by following command 
node server.js

Open google chrome and enter following URL 
http://localhost:8081/index.html
